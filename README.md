# PointMarket Backend

Express + Drizzle backend for the virtual-points prediction market platform.

## Setup

Create your local environment file:

```bash
copy .env.example .env
```

Install dependencies:

```bash
npm install
```

On Windows PowerShell, if `npm` is blocked by script policy:

```bash
npm.cmd install
```

## Run

```bash
npm run dev
```

Or on Windows PowerShell:

```bash
npm.cmd run dev
```
